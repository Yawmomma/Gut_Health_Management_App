function enableDragSort() {
    const items = document.querySelectorAll('.chapter-item');

    items.forEach(item => {
        item.draggable = true;

        item.addEventListener('dragstart', (e) => {
            item.classList.add('dragging');

        const dragging = document.querySelector('.dragging');

            chaptersList.appendChild(dragging);

    });

function getDragAfterElement(container, y) {
    const draggableElements = [...container.querySelectorAll('.chapter-item:not(.dragging)')];

    return draggableElements.reduce((closest, child) => {
        const box = child.getBoundingClientRect();
        const offset = y - box.top - box.height / 2;

        if (offset < 0 && offset > closest.offset) {
            return { offset: offset, element: child };
        } else {
            return closest;
        }
    }, { offset: Number.NEGATIVE_INFINITY }).element;
}

# Education Markdown Management Guide

## Overview

The Education section now supports full markdown file management with:
- Drag-and-drop upload
- Inline markdown editing
- Chapter reordering
- Automatic HTML conversion
- Compact table of contents

---

## Features Added (2026-01-20)

### 1. Drag-and-Drop Upload 📤

**Location**: Left sidebar of `/education/` page

**How to Use**:
1. Drag a `.md` file into the upload zone, OR
2. Click the upload zone to browse for files
3. Enter a section/topic when prompted (e.g., "FODMAPs", "Histamine", "General")
4. File is automatically uploaded and converted to HTML

**What Happens**:
- First `# Heading` becomes the chapter title
- Markdown is converted to styled HTML
- Chapter number is auto-assigned
- Original markdown source is saved for editing

---

### 2. Chapter Reordering 🔄

**Location**: Left sidebar button "Enable Reordering"

**How to Use**:
1. Click "Enable Reordering" in sidebar
2. Drag-and-drop chapters using the grip handles
3. Click "Save New Order" to commit changes
4. Click "Cancel" to discard changes

**What Happens**:
- Chapter numbers are automatically renumbered
- Order persists in database
- All chapter links are disabled during reordering

---

### 3. Markdown Editor ✏️

**Location**: Click "Edit" button (pencil icon) on any chapter

**Features**:
- Full markdown syntax support
- Live title extraction from first `# Heading`
- Section/topic categorization
- Original markdown preserved

## Best Practices

### Writing Chapters

1. **Always start with a level 1 heading**:
   ```markdown
   # Chapter Title Here
   ```

2. **Use semantic heading levels**:
   ```markdown
   # Main Chapter
   ## Section
   ### Subsection
   ```

3. **Include section metadata**:
   - Categorize by topic (FODMAPs, Histamine, etc.)
   - This enables topic filtering

### File Naming

Suggested convention:
```
01-introduction-to-fodmaps.md
02-fructose-explained.md
03-histamine-basics.md
```

## Troubleshooting

### Upload Not Working

**Problem**: File won't upload
**Solution**: Ensure file has `.md` extension

### Markdown Not Rendering

**Problem**: Content shows as plain text
**Solution**:
1. Check if markdown2 is installed: `pip install markdown2`
2. Verify first line has `# Title` heading

### Chapter Order Lost

**Problem**: Order resets after edit
**Solution**: Use the reorder feature after editing to fix numbering
